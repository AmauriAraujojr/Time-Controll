Painel de Controle de Rastreamento de Tempo

Este é um projeto desenvolvido como resposta a um desafio de codificação de front-end proposto pelo Frontend Mentor. O objetivo do desafio é construir um painel de controle de rastreamento de tempo e torná-lo o mais fiel possível ao design fornecido.

Demonstração
Você pode visualizar a demonstração do projeto aqui.

Visão Geral
O Painel de Controle de Rastreamento de Tempo permite aos usuários monitorar e visualizar estatísticas de tempo diárias, semanais e mensais. Ele oferece uma interface interativa e responsiva, permitindo uma experiência consistente em diferentes dispositivos.

Funcionalidades
Visualização do layout otimizado para diferentes tamanhos de tela
Estados de hover para elementos interativos
Alternância entre visualização de estatísticas diárias, semanais e mensais
Exibição dinâmica do texto para o período de tempo anterior, dependendo do intervalo de tempo ativo

Tecnologias Utilizadas
Next.js
TypeScript
Saas

Como Executar o Projeto Localmente
Clone este repositório para o seu ambiente local.
Certifique-se de ter o Node.js e o npm instalados em sua máquina.
Instale as dependências do projeto executando o comando npm install.
Inicie o servidor de desenvolvimento local executando o comando npm run dev.
Abra seu navegador da web e acesse http://localhost:3000 para visualizar o projeto.
